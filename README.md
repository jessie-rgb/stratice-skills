# Stratice Skills (skills.sh)

Reusable skills for Account Executive workflows in IT/Engineering staffing.

## Skills in this repo
- `stratice-outreach-personalizer` — Personalized InMail/Email/Mobile messaging for hiring managers.
- `stratice-talent-gap-analyzer` — Condenses a JD into gaps, market reality, and a delivery plan.

- `stratice-prospect-snapshot-builder` — Bullhorn-ready prospect research snapshot.
- `stratice-candidate-submittal-packager` — Polished client-ready candidate submission pack.
